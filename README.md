# LoL Spell Checker

A lightweight web tool to track enemy summoner spell cooldowns in League of Legends.

## Features

- Track 5 enemy champions (Top, JG, Mid, AD, Sup)
- 3 spell slots per champion (180s, 240s, Flash)
- Cooldown reduction toggles: Cosmic Insight (+18 haste), Ionian Boots (+10 haste)
- Real-time countdown + ETA display
- Game timer
- EN/KR language toggle

## Usage

Open `index.html` in a browser or visit [spell.kkg-dev.com](https://spell.kkg-dev.com).

## Tech

Single `index.html` — no dependencies, no build tools. Deployed via Cloudflare Pages.
