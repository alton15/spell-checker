# LoL Spell Checker - Design Spec

## Overview

League of Legends 소환사 주문 쿨다운 추적 웹 도구. 5명의 적 챔피언 스펠 사용 시점을 기록하고, 쿨다운 감소 옵션에 따라 복귀 시간을 계산하여 표시한다.

## Tech Stack

- **순수 HTML + CSS + Vanilla JS** (단일 `index.html`)
- 빌드 도구 없음, 프레임워크 없음
- 배포: Cloudflare Pages (도메인: spell.kkg-dev.com)

## Core Features

### 1. 게임 타이머

- 게임 시작 버튼 클릭 → 0:00부터 1초 단위로 증가
- 게임 시작 후 버튼이 "게임 종료" 로 변경
- 게임 종료 클릭 → 모든 상태 초기화 (타이머, 쿨다운, 토글)
- 표시 형식: `MM:SS` (00:00). 100분 이상 시 `MMM:SS` 로 확장.
- 게임 시작 전 스펠 버튼 비활성화 (disabled)
- 게임 종료 시 확인 없이 즉시 초기화 (심플하게 유지)

### 2. 프로필 (5개)

- 가로 배치 (5 컬럼)
- 라벨: 탑, 정글, 미드, 원딜, 서폿 (KR) / Top, JG, Mid, AD, Sup (EN)
- 각 프로필은 독립적으로 동작

### 3. 쿨다운 감소 토글 (프로필당 2개)

- **우통** (우주적 통찰력 / Cosmic Insight): 소환사 주문 가속 +18
- **쿨감신** (명석함의 아이오니아 장화 / Ionian Boots of Lucidity): 소환사 주문 가속 +10
- 중복 체크 가능
- 쿨다운 진행 중 토글 변경 → 현재 쿨다운에 영향 없음, 다음 클릭에 반영

### 4. 스펠 버튼 (프로필당 3개)

| 스펠 | 기본 쿨다운 | KR 라벨 | EN 라벨 |
|------|-----------|---------|---------|
| 180초 스펠 | 180s | 180초 | 180s |
| 240초 스펠 | 240s | 240초 | 240s |
| 점멸 | 300s | 점멸 | Flash |

### 5. 쿨다운 계산

공식: `쿨다운 = 기본 쿨다운 × (100 / (100 + 가속))`

| 옵션 | 가속 | 300s | 240s | 180s |
|------|------|------|------|------|
| 없음 | 0 | 300s (5:00) | 240s (4:00) | 180s (3:00) |
| 쿨감신만 | 10 | 272s (4:32) | 218s (3:38) | 163s (2:43) |
| 우통만 | 18 | 254s (4:14) | 203s (3:23) | 152s (2:32) |
| 둘 다 | 28 | 234s (3:54) | 187s (3:07) | 140s (2:20) |

쿨다운은 소수점 이하 버림 (`Math.floor`).

### 6. 쿨다운 표시

스펠 버튼 클릭 시 버튼이 쿨다운 표시로 변경:
- **쿨 / CD**: 남은 쿨다운 카운트다운 (MM:SS)
- **활성 / ETA**: 스펠 복귀 게임 시간 (MM:SS)
- 쿨다운 완료 시 다시 스펠 버튼으로 복원
- 쿨다운 중 재클릭 → 쿨다운 리셋 (현재 토글 상태 기준으로 새로 시작)
- 버튼과 쿨다운 표시 높이 동일 (72px 고정)

### 7. 다국어 (EN/KR)

- 하단에 EN / KR 두 개 버튼 (라디오 스타일, 활성 버튼 하이라이트)
- 전환 대상: 포지션 라벨, 토글 라벨, 스펠 버튼 텍스트, 쿨다운 라벨, 헤더 텍스트, 게임 시작/종료 버튼
- 기본 언어: KR
- 언어 설정은 state에 포함 (`currentLanguage: 'kr' | 'en'`)

## Toggle Visual States

- **비활성**: border #555, 배경 transparent
- **활성**: border #e94560, 배경 #e94560 (체크 표시)

## Color Scheme

- 배경: #1a1a2e (다크 네이비)
- 헤더/풋터: #16213e
- 포지션 라벨/강조: #e94560 (레드)
- 스펠 버튼 (180/240): #533483 (퍼플)
- 스펠 버튼 (점멸): #e94560 (레드)
- 쿨다운 배경: #2a1a3e, 테두리: #ffd700 (골드)
- 남은 시간: #ffd700 (골드)
- 복귀 시간: #4fc3f7 (하늘색)

## State Management

각 프로필의 상태:
```
{
  cosmicInsight: boolean,    // 우통 토글
  ionianBoots: boolean,      // 쿨감신 토글
  spells: [
    { baseCooldown: 180, endTime: null },
    { baseCooldown: 240, endTime: null },
    { baseCooldown: 300, endTime: null }
  ]
}
```

- `endTime`: 스펠 복귀 게임 시간 (초). null이면 대기 상태.

글로벌 상태:
```
{
  gameStartTime: number | null,  // Date.now() 기준, null이면 미시작
  currentLanguage: 'kr' | 'en',
  profiles: [ /* 5개 프로필 */ ]
}
```
